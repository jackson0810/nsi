Containers
This examples illustrates the concept of "Containers." Containers are special types of select boxes in which every item in the select box is part of the value. Notice how when the "Debug" button is clicked, any item in the "Members" select box will be included with the value. Also, when the form is submitted, the value returned to the server will include all the items in the select box. This example also demonstrates the "transferTo()" and "transferFrom()" methods.
